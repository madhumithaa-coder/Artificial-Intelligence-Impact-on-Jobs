Artificial-Intelligence-Impact-on-Jobs :

This project uses Python and machine learning to analyze how artificial intelligence may affect jobs by 2030. It explores automation trends, job role vulnerability, and future workforce impact using data visualization and an interactive Streamlit app.


Project Overview :

This project analyzes the potential impact of Artificial Intelligence on jobs by the year 2030. Using data analysis and machine learning techniques, it identifies how automation may influence different job roles, industries, and workforce demand. The project also provides visual insights through an interactive Streamlit web application.

Objectives :

* Understand how AI and automation affect employment trends
* Analyze job roles most vulnerable to automation
* Predict future job demand using machine learning models
* Present insights through interactive visualizations

Technologies Used :

* Python
* Pandas & NumPy – Data processing
* Matplotlib & Seaborn – Data visualization
* Scikit-learn– Machine learning models
* Streamlit – Web application interface

Features :

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Machine learning-based predictions
* Interactive web dashboard using Streamlit

How to Run the Project :

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```
2. Install required libraries:

   ```bash
   py -m pip install -r requirements.txt
   ```
3. Run the Streamlit app:

   ```bash
   py -m streamlit run AIIMPACT.py
   ```

Output :

* Visual insights on AI-driven job impact
* Predictive analysis of job trends
* User-friendly dashboard for exploration
